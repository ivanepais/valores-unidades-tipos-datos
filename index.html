Capitulo 8: valores y unidades

Todas las propiedades de CSS tienen un conjunto de valores que admiten. En la documentacion las propiedades y los valores denotan distinto.

Colores

Hay muchas formas de especificar los colores, con palabras claves o funciones, etc. El sistema de color estadar es de 24 bits, permite 16,7 millores de colres distintos a partir de una combinacion de diferentes canales de rojo, verde y azul con 256 valores diferentes por canal.

Palabras claves

Hay una multitud de palabras claves para especificar colores.

Hexadecimales RGB

Son codigos que empiezan con un hastag(#) seguido de seis cifras hexadecimales, cada cifra puede tomar 16 valores que se encuentran entre 0 y. Cada par de cifra representa uno de los canales (rojo, verde, azul) que nos permite especificar los 256 colores por canal.

RGB y RGBA

Un color RGB es una funcion, rgb (); tiene tres parametros que representa los colores rojo, verde y azul. Cada canal se representa por un numero decimal entre 0 y 255.

Los colores RGBA funcionan como el anterior pero agrega un canal alfa de color que controla la opacidad su valor se establece entre 0 y 1 (trasparente/opaco). Los valores intermedios son diferentes niveles de trasparencia. Notación: despues de la coma se deja un espacio se pone un punto y el valor. A diferencia de la propiedad opacity, un color RGBA solo aplica al elemento que especificas y no a todo el contenedor. Ej: si se lo aplicas a un fondo solo va a ser trasparente ese fondo y no el contenido/texto.

HSL y HSLA

Es una funcion como rgb pero en lugar de los valores para rojo, verde y azul, HSL admite valores de matriz, saturacion y luminosidad. Matriz: tono base del color, su valor va de 0 y 360 que representa un angulo en torno a una rueda de colores. Saturación: es la intensidad del color, toma un valor entre 0 (tono gris) y 100% (max saturación). Luminosidad: es la claridad o brillo del color, su valor está entre 0 (negro) y 100% (blanco). Tiene su canal alfa con la funcion HSLA que controla la opacidad del elemento.

Los colores por funciones (RGB y HSL) tiene la capacidad de admitir el canal alfa sin especificar la función para este canal y viceversa.

Imagenes

Se usa cuando este tipo de dato es valido en una propiedad. Se puede usar la funcion url() o poner un degradado. Estos son los estandarizados pero hay otros si se consulta la documentación.

Position

Representa un conjunto de coordenadas 2d que se utiliza para colocar un elemento. Puede tomar palabras claves: top, right, button, left y center; alinea el elemento en los limites de la caja, salvo center. Position puede tener dos valores: el primero es para el eje x (horizontal) y el segundo para el eje y (vertical). El elemento se empieza a mover desde la esquina inferior izq.

Cadenas e identificadores

Las palabras claves como las que hemos visto anteriormente se denominan identificadores,las escribimos sin comillas. Una cadena es un texto que se escribe entre comillas que se usa para identificar el contendio que generamos.

Funciones

Este valor es un bloque de codigo reutilizable para ejecutar varias veces una tarea repetitiva. Tambien las hay en CSS, ya hemos visto rgb(), hsl() y url().

La funcione calc() permite hacer calculos para sacar valores que no se pueden describir con palabras claves o simples que no podemos saber de antemano, sobre todo cuando hay elementos relacionados unos con otros.

Conclusión

Cada propiedad tiene una lista definida de valores que admite y cada valor tiene una definicion que explica sus subvalores permitidos.
